# MoodSync-AI

A micro-MVP mood-based content curator using:
- Supabase (DB)
- Express.js (Backend)
- Postman (API testing)
- Simple frontend (HTML/JS or React)

## Project Structure
- backend/ – Express server
- frontend/ – UI
- supabase/ – schema.sql + seed.sql
- postman/ – Postman collection
- docs/ – diagrams & notes
- assets/ – screenshots & images

## Setup
1. Clone repo
2. Create `.env` (copy `.env.example`)
3. Install backend dependencies
4. Start server
5. Test with Postman

## Goal
Show both CSE & PM skills in a clean, small project.
